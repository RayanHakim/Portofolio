# 🚀 Personal Portfolio - Rayan Luqman Hakim

Selamat datang di repositori portofolio saya! Project ini dibangun menggunakan **React.js** dan **Tailwind CSS**, serta dideploy secara otomatis menggunakan **GitHub Pages**. Portofolio ini dirancang untuk menampilkan rekam jejak saya sebagai lulusan Teknik Informatika yang berfokus pada *Software Development*, *Machine Learning*, dan *Quality Assurance*.

🌐 **Live Demo:** [https://rayanhakim.github.io/Portofolio/](https://rayanhakim.github.io/Portofolio/)

---

## 🌟 Proyek Utama

### 1. 🇮🇩 Pahlawan Indonesia Mobile App
Aplikasi edukasi berbasis Flutter yang mengintegrasikan:
* **Autentikasi**: Login & Register terenkripsi.
* **Database**: SQLite untuk manajemen data lokal pahlawan.
* **Hardware Integration**: Sensor Gyroscope dan Sistem Notifikasi lokal.
* **Tech**: Flutter, Dart, REST API.

### 2. 📊 Sentiment Analysis Dashboard
Aplikasi desktop berbasis Python untuk analisis sentimen otomatis:
* **Model**: Hybrid Ensemble (Naive Bayes & Logistic Regression).
* **NLP Pipeline**: Normalisasi slang, penanganan emoji, dan negasi.
* **Tech**: Python, Scikit-Learn, Tkinter, NLTK.

### 3. 🧪 QA Automation & Pentest
Dokumentasi pengujian menyeluruh pada Web Portal pemerintahan:
* **Automation**: UI Testing (Selenium), Unit Testing (Pytest), dan Load Testing (Locust) hingga 1000 users.
* **Cybersecurity**: Audit keamanan terhadap Brute Force, XSS, dan SQL Injection.
